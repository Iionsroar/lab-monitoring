## Frontend
Install dependencies
```sh
npm install
```

Start the app in development mode (hot-code reloading, error reporting, etc.)
```sh
quasar dev
```

Lint with [ESLint](https://eslint.org/)
```sh
npm run lint
```

Format the files

```sh
npm run format
```

Build the app for production

```sh
quasar build
```
